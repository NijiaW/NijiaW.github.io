# Simona's Profile Website

Welcome to my personal profile website! This site showcases my skills, projects, and professional background, and it also serves as a portfolio for potential employers or collaborators.

## Features

- **About Me**: A brief introduction to my background, experience, and personal interests.
- **My Skillsets**: An overview of my technical skills, including programming languages, tools, and technologies I work with.
- **My Projects**: Descriptions of key projects I have worked on, showcasing my work in AI, web development, and data analytics.
- **My Habits**: A personal section featuring my interests and hobbies, including travel photography.
- **Contact**: Information on how to get in touch with me for job opportunities, collaborations, or just to connect.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome** for icons
- **GitHub Pages** for hosting
